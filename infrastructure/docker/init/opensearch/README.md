# OpenSearch Initialization

No index templates, mappings, or indexes are created at this
infrastructure-only milestone — Enterprise Search indexing begins in
Phase 7 of `docs/architecture/specification/110_Implementation_Roadmap.md`.

This directory is reserved for the future initialization script that will
create the base index templates once the Enterprise Search Domain's
implementation begins (`docs/architecture/specification/70_Enterprise_Search.md`
onward).
