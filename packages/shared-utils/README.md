# @cerebrum/shared-utils

Small, generic, dependency-free TypeScript utility functions shared across
TypeScript packages/apps — no business logic, no domain awareness.

## Public Interfaces

`src/index.ts` — currently empty.

## Dependencies

None (deliberately — dependency-free is this package's defining
constraint).

## Limitations

Empty at Repository Foundation. If a proposed addition requires a
dependency or encodes a business rule, it belongs elsewhere — see
`docs/architecture/naming-conventions.md`.
